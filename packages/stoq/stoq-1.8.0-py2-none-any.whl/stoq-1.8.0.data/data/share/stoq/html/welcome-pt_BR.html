Sobre o Stoq

Bem-vindo ao Stoq!

Stoq está configurado em modo de demonstração, para que você possa fazer um test-drive.

Durante o uso do modo de demonstração, o plugins ECF estará habilitado e utilizará um emulador de impressora fiscal para que o usuário tenha a sensação de como é o sistema em produção.

Após a instalação e a habilitação do modo de produção, o emulador e limitações são removidos.

Caso seja solicitado o nome do usuário e senha, estes são:

  • usuário: admin
  • senha: admin

Contato Comercial

Para tratar assuntos comerciais, fazer um primeiro contato para uma proposta nova ou agendar uma visita, entre em contato com nosso departamento comercial <comercial@stoq.com.br>.

Contato Técnico

Para contactar a equipe do projeto Stoq, ou para encaminhar dúvidas, sugestões ou soluções utilize as listas de discussão.

Para tratar de assuntos técnicos como desenvolvimento, arquitetura e usabilidade, é recomendado falar com o responsável pelo projeto, Christian Reis <comercial@stoq.com.br>.

Caso possua uma conexão à internet disponível, você pode entrar em contato pelo nosso chat online.
