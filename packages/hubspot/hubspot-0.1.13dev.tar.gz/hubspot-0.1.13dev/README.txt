====================
HubSpot Api Wrapper
===================

HubSpot API Wrapper for python. not a complete package

To Install for Development testing:
pip install hubspot --pre

to upgrade package
pip install hubspot --pre --upgrade



Usage:

portal = HubSpot(portal_id="YourPortal", hapikey="YOUR_HAPI_KEY")

Modules:
portal.files
portal.contacts
portal.properties
portal.form 

More to come.
